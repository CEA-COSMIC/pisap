pysap

pySap is a Python package related to sparsity and its application in astronomical data analysis. This package is based on the 'sparse2d' C++ library that allows sparse decomposition, denoising and deconvolution.

pysap.base

This module defines common structures for astronomical and neuroimaging datasets.

pysap.plugins

This module defines all the declared plugins.

pysap.apps

Module that provides a graphical application to execute functions.

pysap.extensions

This module contains a wrapping of the 'sparse2d' C++ library that allows fast sparse decomposition, denoising and deconvolution.

pysap.plotting

This module defines common vizualisation tools.

pysap.base.loaders

This module defines common loaders to deal with astronomical and neuroimaging datasets.

pysap.plugins.astro

This module defines all the Astro related plugins.

pysap.plugins.mri

This module defines all the MRI related plugins.

pysap.plugins.astro.deconvolve

This module defines all the Astro deconvolution plugins.

pysap.plugins.mri.parallel_mri

This module defines all the MRI reconstruction plugin.

pysap.plugins.mri.reconstruct

This module defines all the MRI reconstruction plugin.
